# ☕ wayd - Take quick breaks with other programmers

[![](https://img.shields.io/badge/Download-wayd-blue.svg)](https://github.com/gilburtgradatory270/wayd)

wayd helps you manage your focus during long coding sessions. It provides a simple feed where you see what other developers post while they deal with difficult bugs. You stay connected to your community without leaving your workflow or dealing with complex account setups.

## 📁 What is wayd?

Programming requires deep focus, but everyone needs a brief pause to reset. wayd connects to GitHub Issues to show you what others face in real time. You read short updates, share reactions, and return to your code. 

The software works without a central server. You do not need to create an account or provide personal data. It uses your existing GitHub connection to pull data and display it in a simple window.

## 💡 How to Use the Feed

When you open wayd, you see a stream of posts. These posts come from active GitHub Issues.

## 💬 Frequently Asked Questions

**Does wayd store my password?** 
No, the application uses your browser authentication to access GitHub information. 

**Can I use wayd on a Mac?** 
This version is designed specifically for Windows machines. 

**Does the app notify me when I have new messages?** 
wayd stays quiet. It only fetches data when you open the window. You choose when to take a break.

**Can I hide my posts later?** 
Your posts are public on the GitHub Issues tracked by the app. If you want to remove a post, you can delete it directly through your GitHub account. 

**Why does the feed look blank sometimes?** 
If no one is posting on the tracked issues, the feed appears empty. This is normal behavior. 

**Does it cost money to use?** 
The software is free for every user. 

**How do I uninstall the app?** 
Open your Windows Settings, go to Apps, find wayd in the list, and select Uninstall. This removes all program files from your system.
